# 6D-RRT: Parallel Perceptual Relational Training

## Purpose
The **6D-RRT** application is a high-bandwidth cognitive training tool designed to transition the brain from **serial symbolic processing** to **parallel perceptual mapping**. By offloading complex relational logic onto the visual and auditory cortex, the system aims to expand working memory (WM) capacity and enhance relational reasoning (RRT) through "active friction."

Unlike traditional training, this protocol forces the **Executive Control Network (ECN)** to maintain and invert up to six independent variables simultaneously, effectively training the user to navigate a **Relational Hyper-Space**.

---

## Function

### 1. Multi-Dimensional Encoding
The application maps abstract relational logic to intuitive environmental cues. A single stimulus represents a 6-tuple vector:

| Dimension | Variable | Visual/Auditory Marker |
| :--- | :--- | :--- |
| **1 & 2: XY Plane** | Spatial | Screen Position (e.g., North East) |
| **3: Vertical** | Value ($Z$) | **Lightness:** Lighter (Above) / Darker (Below) |
| **4: Temporal** | Hue ($T$) | **Color:** Green (Future) / Red (Past) |
| **5: Magnitude** | Font Size ($M$) | **Intensity:** large (Bigger) / small (Smaller) |
| **6: Hierarchy** | Frequency ($H$) | **Pitch:** High (Superior) / Low (Inferior) |
